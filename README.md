# HR-Analytics
A data-driven HR Analytics project using Power BI to identify key factors behind employee turnover and satisfaction


HR Analytics Dashboard
This project features a comprehensive HR Analytics Dashboard built in Power BI. It provides a deep dive into employee data to uncover trends related to attrition, workforce demographics, and job satisfaction, enabling data-driven decision-making for HR departments.

Project Overview
The goal of this project is to help HR managers identify key factors contributing to employee turnover (attrition) and understand the overall health of the organization's workforce.

By analyzing metrics like education background, age bands, and job roles, the dashboard highlights which segments of the workforce are most at risk and where engagement efforts should be focused.

Dataset Information
The analysis is based on a dataset containing 40+ attributes for each employee, including:

Demographics: Age, Gender, Marital Status, Education Field.

Employment Details: Department, Job Role, Monthly Income, Years at Company, Overtime status.

Satisfaction Levels: Environment Satisfaction, Job Involvement, Job Satisfaction, Work-Life Balance.

Attrition: Whether the employee has left the company or is still active.


Key KPIs & Metrics
Total Employee Count: Overall workforce size.

Attrition Rate: Percentage of employees who have left the organization.

Average Age: Demographic snapshot of the workforce.

Average Monthly Income: Financial overview across different roles.

Work-Life Balance & Job Satisfaction: Qualitative metrics to gauge employee morale.


Key Insights Explored
Attrition by Department: Which departments (e.g., Sales, R&D, HR) experience the highest turnover?

Education Field Impact: Does the employee's background influence their longevity at the company?

Age Band Analysis: Identification of turnover trends across different age groups (e.g., 25-34 vs. 45-54).

Overtime vs. Attrition: Correlation between working extra hours and the likelihood of leaving.

Role-Specific Trends: Highlighting turnover rates for specific positions like Laboratory Technicians or Sales Executives.


Tools Used
Power BI: Data visualization, DAX measures, and dashboard design.

Excel/CSV: Initial data storage and cleaning.


Files in this Repository
HR DASHBOARD.pbix: The interactive Power BI dashboard file.

HR Data.xlsx - HR data.csv: The raw dataset used for the analysis.


How to View the Project
